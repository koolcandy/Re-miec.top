德摩根定律（对偶律）

$\overline{A+B} = \overline{A}\times\overline{B}$
$\overline{A\times B} = \overline{A}+\overline{B}$

即：

$\neg(p \wedge q) \equiv(\neg p) \vee(\neg q)$
$\neg(p \vee q) \equiv(\neg p) \wedge(\neg q)$

AB + A'C + BC
= AB + A'C + (1)BC
= AB + A'C + (A + A'))BC
= AB + A'C + (ABC + A'BC)

= AB(1 + C) + A'C(1 + B)
