一点名词
----


* databases（数据库）
* relational databases（关系数据库）
* database management systems（数据库管理系统，DBMS）
* relational database management systems（关系数据库管理系统，RDBMS）。关系数据库是将数据间的关系以数据库表的形式加以表达，并将数据存储在表格中，以便于查询。
